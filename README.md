# 🚗 环境可持续标准管理系统

一个强大的环境标准管理和对比分析工具

## ✨ 功能特性

### 📋 标准管理
- **在线搜索**：智能搜索标准文档
- **手动添加**：自定义添加标准信息
- **文档上传**：支持PDF和Word文档，自动提取关键信息
- **分类管理**：腐蚀、老化、ELV、低碳四大类别

### 📄 文档处理
- **智能提取**：自动识别标准编号、发布日期、适用范围等
- **文档查看**：
  - PDF文档：浏览器内直接预览
  - Word文档：下载后查看
  - 在线链接：直接跳转原网页

### 📊 标准对标
- **多标准对比**：同时对比多个标准
- **差异识别**：自动高亮关键信息差异
- **智能分析**：相似度分析和关键差异提取
- **报告导出**：导出对比分析报告

### 💾 数据管理
- **本地存储**：数据保存在浏览器LocalStorage
- **导入导出**：支持JSON格式数据备份
- **批量操作**：批量添加、删除标准
- **搜索过滤**：快速查找目标标准

## 🚀 快速开始

### 在线使用（最简单）

直接访问在线版本，无需安装任何软件！

### 本地使用

1. 下载 `index.html` 文件
2. 双击打开即可使用
3. 所有功能离线可用

## 📖 使用指南

### 添加标准的三种方式

#### 方式1：在线搜索
1. 点击"在线搜索"标签
2. 输入关键词搜索
3. 从结果中选择需要的标准
4. 点击"添加到数据库"

#### 方式2：手动添加
1. 点击"手动添加"标签
2. 填写标准信息
3. 可添加文档链接或本地路径
4. 点击"添加标准"

#### 方式3：文档上传
1. 点击"文档上传"标签
2. 拖拽或选择PDF/Word文档
3. 系统自动提取信息
4. 文档完整保存，可直接查看

### 标准对标分析

1. 点击"标准对标"标签
2. 选择要对比的标准（至少2个）
3. 点击"开始智能对标分析"
4. 查看对比结果和差异分析

### 查看文档

- **上传的文档**：点击"查看上传文档"按钮
  - PDF：浏览器内直接预览
  - Word：下载后用Office打开
- **在线链接**：点击"打开在线链接"按钮跳转

## 🛠️ 技术栈

- **前端框架**：纯HTML/CSS/JavaScript
- **UI框架**：Bootstrap 5
- **图标**：Bootstrap Icons
- **PDF处理**：PDF.js
- **Word处理**：Mammoth.js
- **数据存储**：LocalStorage

## 📦 部署到外网

### 方式1：Netlify（最简单，5分钟完成）

1. 访问 https://app.netlify.com/drop
2. 将包含 index.html 的文件夹拖到页面
3. 自动生成访问链接
4. 完成！

### 方式2：GitHub Pages

1. Fork本仓库或创建新仓库
2. 上传 index.html 文件
3. 在仓库设置中启用GitHub Pages
4. 选择main分支
5. 访问生成的链接

### 方式3：Vercel

1. 注册Vercel账号
2. 导入GitHub仓库
3. 自动部署
4. 访问生成的链接
